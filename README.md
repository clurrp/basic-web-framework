

# Build Webframework

- Create webframework, including event handling

## Table of Contents
- Clone repo
- Install Dependencies
- Start Project
- View Application
- Acknowledgements

### Clone repo

` > git clone `

### Install Dependencies

 `> npm i`

### Start Project

  `> json-server -w db.json`
  `> npx parcel index.html`

### View Application

- http://localhost:1234

### View JSON Server

- - http://localhost:3000/users
